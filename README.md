# MesclaInvest

Projeto desenvolvido para a disciplina **Projeto Integrador 3 (PI3)** — Engenharia de Software — PUC-Campinas (2026).


## Descrição

O MesclaInvest é uma aplicação mobile que simula um ambiente de investimento em startups por meio da negociação de tokens digitais.

O sistema permite ao usuário visualizar startups, consultar informações relevantes e realizar operações simuladas de compra e venda de tokens.

Todas as operações possuem caráter exclusivamente acadêmico.



## Arquitetura

O sistema é dividido em três camadas:

- **Frontend:** Flutter (Dart)
- **Backend:** Node.js
- **Banco de Dados:** Firebase Firestore



## Funcionalidades

- Autenticação de usuários (cadastro, login, recuperação de senha, 2FA)
- Catálogo de startups com visualização de detalhes
- Negociação simulada de tokens (compra e venda)
- Gestão de carteira e histórico de transações
- Dashboard com visualização de valorização



## Regras de Negócio

- Validação de saldo
- Atualização de preço baseada em transações
- Cálculo de valorização dos tokens
- Registro e consistência das operações

## Escopo

O sistema não contempla:

- Transações financeiras reais  
- Integração com bancos ou pagamentos  
- Emissão de tokens em blockchain
